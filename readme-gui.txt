
C2Onto GUI - код на Си в триплеты, триплеты в БД на Stardog-сервер, а затем экспорт с сервера в онтологию формата turtle.



1. Установить зависимости

$ pip install pystardog pycparser

2. onto-creator.py создаёт схему онтологии - c_schema_2020-01.rdf
	(можно не запускать, т.к. схема прилагается)
	Требует зависимости (можно не устанавливать):
	$ pip install owlready2


3. Порядок работы.

3.0 Запускаем GUI - c2onto_gui.py
  Для работы требуются файлы:
	c_schema_2020-01.rdf
	c2onto.py

3.1 Кликаем зелёную кнопку справа внизу
3.2 Ещё раз
3.3 Нажимаем красную кнопку Upload

	Должна создаться БД на сервере (если он запущен),
	И загрузиться триплеты (см. лог в консоли)

3.4 При успехе разблокируется кнопка Download - нажимаем

	Должна скачаться онтология в файл exported_onto.ttl
